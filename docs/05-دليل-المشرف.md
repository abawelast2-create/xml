# دليل المشرف (Admin)

---

## الدخول للوحة الإدارة

الرابط: `/admin/login.php`  
بيانات الدخول الافتراضية: `admin` / `Admin@1234`

> **غيّر كلمة المرور فوراً** من: الإعدادات ← تبويب الأمان

---

## لوحة التحكم (Dashboard)

تعرض لحظياً:
- **عدد الحاضرين اليوم** — من سجّلوا دخولاً
- **عدد المنصرفين** — من سجّلوا انصرافاً
- **الغائبين** = إجمالي الموظفين المفعّلين − الحاضرين
- **آخر 10 تسجيلات** مع الأسماء والأوقات

## إدارة الموظفين

### إضافة موظف جديد

1. اضغط **"➕ إضافة موظف"**
2. أدخل:
   - **الاسم الكامل** ← يظهر على صفحة الحضور
   - **المسمى الوظيفي** ← يظهر تحت الاسم
   - **PIN** ← رقم سري فريد (للرجوع إليه مستقبلاً)
   - **الفرع** ← اختر الفرع التابع له الموظف (يحدد نطاقه الجغرافي)
   - **رقم الواتساب** (اختياري) ← بصيغة دولية مثل `966501234567`
3. اضغط **"حفظ"** ← يُولّد الرابط الفريد تلقائياً

> **ملاحظة:** إذا اختير فرع للموظف، سيستخدم النظام إحداثيات الفرع ونصف قطره للتحقق من موقعه، بدلاً من الإعدادات العامة.

### إرسال الرابط للموظف

- **مع رقم واتساب:** اضغط زر **"💬 واتساب"** ← يفتح رسالة جاهزة
- **بدون رقم:** اضغط **"📋 نسخ"** ← انسخ الرابط وأرسله يدوياً

### إجراءات أخرى على الموظف

| الزر | الوظيفة |
|------|---------|
| ✏️ | تعديل الاسم والوظيفة والهاتف |
| 🔄 | إعادة توليد رابط جديد (الرابط القديم يُلغى) |
| 🔕 / 🔔 | تعطيل / تفعيل الموظف |
| 🔒 ربط صارم | تفعيل الربط الصارم — يمنع أي جهاز مختلف |
| 👁️ ربط مراقبة | تفعيل المراقبة الصامتة — يسمح لكن يُسجّل التلاعب |
| 📱🔓 | إعادة تعيين بصمة الجهاز ← يُحرر الرابط من جهاز محدد |
| 🗑️ | أرشفة الموظف (حذف ناعم — يمكن استعادته) |

### الإجراءات الجماعية (أعلى الصفحة)

| الزر | الوظيفة |
|------|--------|
| **تجديد جميع الروابط** | يولّد روابط جديدة لكل الموظفين النشطين |
| **إرسال الكل واتساب** | رسالة واحدة بكل الروابط مرتبة حسب الفرع |
| **🔒 ربط صارم للجميع** | تفعيل الربط الصارم لجميع الموظفين دفعة واحدة |
| **👁️ مراقبة للجميع** | تفعيل المراقبة الصامتة لجميع الموظفين |

### عمود "الجهاز"

| الأيقونة | المعنى |
|----------|--------|
| 🔒 | ربط صارم — الرابط مربوط بجهاز ويُحظر أي جهاز آخر |
| 👁️ | مراقبة صامتة — مربوط لكن يسمح ويُسجّل التلاعب |
| ⏳ ينتظر | وضع الربط مفعّل — سيُربط عند الدخول التالي |
| 🔓 حر | الرابط حر — لا يحتاج ربط جهاز |
| 📱 اسم الموظف | مالك الجهاز — الموظف صاحب أعلى استخدام لنفس الجهاز |

## صفحات التقارير

### تقرير التأخر (`/admin/late-report.php`)
- يُظهر الموظفين الذين تأخروا عن وقت الدخول
- قابل للتصفية حسب التاريخ والفرع والنوبة
- تصدير CSV مع حماية من حقن الصيغ (Formula Injection)

### التقرير اليومي (`/admin/report-daily.php`)
- ملخص يومي شامل للحضور والانصراف والغياب
- قابل للتصفية حسب التاريخ والفرع والنوبة
- يُظهر وقت الدخول والانصراف وساعات العمل ودقائق التأخير

### التقرير الشهري (`/admin/report-monthly.php`)
- ملخص شهري لكل موظف: أيام الحضور والغياب ومتوسط التأخير
- قابل للتصفية حسب الشهر والفرع والنوبة
- تصدير CSV

### تقرير الانصراف المبكر (`/admin/report-early.php`)
- يُظهر الموظفين الذين انصرفوا قبل نهاية الدوام
- إحصائيات: عدد المتميزين، أيام التبكير، إجمالي الدقائق
- ترتيب حسب الأكثر تبكيراً
- تصدير CSV

### تقرير الغياب (`/admin/report-absence.php`)
- يُظهر الموظفين الغائبين خلال فترة محددة
- قابل للتصفية حسب التاريخ والفرع والنوبة
- إحصائيات: إجمالي أيام الغياب ومعدل الغياب

### مقارنة الفروع (`/admin/report-branches.php`)
- مقارنة بين أداء الفروع من حيث الحضور والغياب والتأخير
- رسم بياني للمقارنة
- ترتيب الفروع حسب الأداء

### التقارير البيانية (`/admin/report-charts.php`)
- رسوم بيانية تفاعلية للحضور والغياب
- إحصائيات بصرية شاملة

## إدارة الإجازات (`/admin/leaves.php`)

### عرض الإجازات
- قائمة جميع طلبات الإجازات مع الفلترة حسب الحالة والنوع
- حالات الإجازة: **معلّقة** / **موافق عليها** / **مرفوضة**

### إضافة إجازة
1. اختر الموظف
2. حدد النوع: سنوية / مرضية / طارئة / أخرى
3. حدد تاريخ البداية والنهاية
4. أضف السبب (اختياري)

### الموافقة/الرفض
- اضغط ✅ للموافقة أو ❌ للرفض
- يمكن إضافة ملاحظات المشرف

## نقل الموظفين (`/admin/employee-transfer.php`)

- تحويل موظف من فرع لآخر
- يُسجّل في سجل المراجعة (Audit Log) تلقائياً
- يُحدّث الفرع في بيانات الموظف فوراً

## أرشيف الموظفين (`/admin/employees-archive.php`)

- عرض الموظفين المحذوفين (حذف ناعم)
- إمكانية استعادة الموظف المحذوف
- عرض تاريخ الحذف وبيانات الموظف

---

## الإشعارات (`/admin/notifications.php`)

- عرض جميع الإشعارات مع مؤشر الإشعارات غير المقروءة
- أنواع الإشعارات: معلومات / تحذير / خطر / نجاح
- تعيين كمقروءة فردياً أو دفعة واحدة
- عدد الإشعارات غير المقروءة يظهر في الشريط العلوي

## سجل المراجعة (`/admin/audit-log.php`)

- تتبع جميع عمليات الإدارة: إضافة/تعديل/حذف الموظفين، تغيير الإعدادات، إلخ
- يُسجّل: نوع العملية، الوصف، المشرف المُنفّذ، عنوان IP، الوقت
- قابل للتصفية والبحث

---

## النسخ الاحتياطي (`/admin/backups.php`)

### إنشاء نسخة احتياطية
- اضغط "إنشاء نسخة احتياطية" ← يتم تصدير جميع الجداول كملف SQL
- الملفات تُحفظ في مجلد محمي على السيرفر

### استعادة نسخة احتياطية
- اختر ملف النسخة الاحتياطية واضغط "استعادة"

### تنزيل/حذف النسخ
- تنزيل النسخة كملف SQL
- حذف النسخ القديمة

---

## تتبع الوثائق (`/admin/documents-expiry.php`)

- متابعة انتهاء صلاحية وثائق الموظفين (هوية، رخصة، إقامة، إلخ)
- تنبيه عند اقتراب انتهاء الصلاحية

---

## حالات التلاعب (`/admin/tampering.php`)
- جميع حالات التلاعب المُكتشفة (جهاز مختلف، تسجيل بالنيابة...)
- يُظهر الخطورة والتفاصيل ومالك الجهاز

### البلاغات السرية (`/admin/secret-reports.php`)
- البلاغات المُرسلة من الموظفين عبر زر البلاغ العائم
- تشمل: نص + صور + تسجيل صوتي (مع مؤثرات)
- حالات: جديد / تمت المراجعة / مُعالِج

## سجلات الحضور

### تصفية السجلات
- حسب **التاريخ** (يوم محدد أو نطاق)
- حسب **الموظف**
- حسب **نوع التسجيل** (دخول / انصراف / إضافي)

### تصدير البيانات
- زر **"📥 تصدير CSV"** ← ملف Excel متوافق
- يصدّر السجلات المفلترة الحالية

## الإعدادات

### تبويب الموقع الجغرافي
- **موقع العمل:** دبّن الموقع على الخريطة التفاعلية أو أدخل الإحداثيات يدوياً
- **نصف قطر النطاق:** المسافة المسموحة بالمتر (الحد الأدنى 1 متر)
  - مبنى صغير: 50-100 متر
  - مجمع: 200-500 متر
  - ميدان واسع: 500-1000 متر

### تبويب أوقات الدوام
| الإعداد | الوصف |
|---------|-------|
| بداية/نهاية الدوام | العرض على الصفحة فقط |
| نافذة تسجيل الدخول | المدة الزمنية المسموح فيها بتسجيل الحضور |
| نافذة تسجيل الانصراف | المدة الزمنية المسموح فيها بتسجيل الانصراف |
| عرض زر الانصراف قبل | كم دقيقة قبل وقت البداية يظهر الزر |

### تبويب الدوام الإضافي
- تفعيل/تعطيل ميزة الدوام الإضافي
- بعد كم دقيقة من الانصراف يبدأ
- الحد الأدنى للمدة

### تبويب الأمان
- تغيير كلمة مرور المشرف (يشترط كلمة المرور الحالية)

## سيناريوهات شائعة

### ربط جهاز موظف لأول مرة
1. الموظف يفتح رابطه → يمكنه التسجيل بدون ربط جهاز
2. عندما تريد ربطه، اختر من القائمة:
   - **🔒 ربط صارم** ← لمنع أي جهاز آخر
   - **👁️ ربط مراقبة** ← للسماح مع تسجيل التلاعب
3. الموظف يفتح الرابط على جهازه ← يُربط الجهاز تلقائياً
4. بعد الربط:
   - **صارم:** لن يعمل الرابط على أي جهاز آخر
   - **مراقبة:** يعمل من أي جهاز لكن التلاعب يُسجّل

> **ملاحظة:** لا يحدث ربط تلقائي — المشرف يتحكم بالكامل.

### الموظف غيّر هاتفه
1. ابحث عن الموظف في قائمة الموظفين
2. اضغط زر **📱🔓** (فك ربط الجهاز)
3. إذا أردت ربط الجهاز الجديد، اختر **🔒 ربط صارم** أو **👁️ ربط مراقبة**
4. الموظف يفتح رابطه على الجهاز الجديد → يُربط تلقائياً

### تجديد وإرسال جميع الروابط
1. اضغط **"تجديد جميع الروابط"** ← يولّد روابط جديدة للجميع
2. اضغط **"إرسال الكل واتساب"** ← يفتح واتساب برسالة شاملة
3. الرسالة تحتوي كل الروابط مرتبة حسب الفرع

### الموظف فقد رابطه
1. من قائمة الموظفين اضغط **🔄** لتوليد رابط جديد
2. أرسل الرابط الجديد عبر واتساب
3. الرابط القديم يُلغى تلقائياً

### فتح الرابط من جهاز مختلف (اشتباه بالتشارك)
- صفحة الموظف ستعرض شاشة القفل 🔒 (إذا كان الجهاز مربوطاً)
- إذا لم يكن مربوطاً، يمكنه الدخول من أي جهاز
- لا يمكنه التسجيل من جهاز آخر مربوط حتى تتدخّل أنت
